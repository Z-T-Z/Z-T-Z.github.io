Hexo第一篇博客,简述Hexo部署过程

曾志
2017-09-14

关于hexo的部署需要使用到的工具:
1.安装Node 作用：用来生成静态页面的.
2.安装Git 作用：把本地的hexo内容提交到github上去.
3.github 帐号 作用：是用来做博客的远程创库、域名、服务器
4.域名

安装部署过程简述:
安装教程在网上可以查到很多,这里不过多赘述,在次简述安装过程中遇到的问题,以及解决方式
一般来说,node,git安装时一路next就行,安装路径默认为C盘,可手动修改路径,具体不多赘述.
这里说说安装命令
在本地新建一个文件夹,进入后选择Git Bash here,可打开git 窗口,
使用安装命令 npm install -g hexo进行安装
之后是初始化 hexo init
生成静态页面 hexo generate 可简化为hexo g


这里是安装完成的文件夹

如果一切顺利,到这里便完成了hexo 本地安装,使用hexo s 命令可启动本地服务,打开http://localhost:4000
可看到本地效果.安装过程中不会遇到很麻烦的问题,只要注意命令以及文件夹,安装完后可在文件夹中找到
